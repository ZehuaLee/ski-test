std::vector<Cycles> MinorFUTiming::srcRegsRelativeLats

Cycle offsets from the scoreboard delivery times of register values for each of this instruction's source registers (in srcRegs order).

The offsets are subtracted from the scoreboard returnCycle times. For example, for an instruction type with 3 source registers, [2, 1, 2] will allow the instruction to issue upto 2 cycles early for dependencies on the 1st and 3rd register and upto 1 cycle early on the 2nd.
