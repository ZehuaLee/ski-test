Detailed Description

This class is a minimal wrapper around SimObject.

It is used to declare a python derived object that can be added as a ProbeListener to any other SimObject.

It instantiates manager from a call to Parent.any. The vector of listeners is used simply to hold onto listeners until the ProbeListenerObject is destroyed.
