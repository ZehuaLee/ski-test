Detailed Description

Implements a local predictor that uses the PC to index into a table of counters.

Note that any time a pointer to the bp_history is given, it should be NULL using this predictor because it does not have any branch predictor state that needs to be recorded or updated; the update can be determined solely by the branch being taken or not taken.
