00040 /*
00041  * These classes define an expression language over uint64_t with only
00042  * a few operators.  This can be used to form expressions for the extra
00043  * delay required in variable execution time instructions.

00045  * Expressions, in evaluation, will have access to the ThreadContext and
00046  * a StaticInst.
00047  */
