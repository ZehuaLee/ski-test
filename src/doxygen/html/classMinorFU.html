Detailed Description

A functional unit that can execute any of opClasses operations with a single op(eration)Lat(ency) and issueLat(ency) associated with the unit rather than each operation (as in src/FuncUnit).

This is very similar to cpu/func_unit but replicated here to allow the Minor functional units to change without having to disturb the common definition.
