Tick I2CBus::write ( PacketPtr  pkt  )  [virtual]

The default i2c bus driver used by the realview pbx board writes to this device one bit at a time.

To facilitate making new i2c devices, i2cBus::write takes care of the low-level details of the i2c protocol. See the I2C Specification [1] for a detailed description of the protocol.

[1] - http://www.nxp.com/documents/user_manual/UM10204.pdf

Implements PioDevice.
