Detailed Description

Event wrapper for dmaDone().

This event call pushes its this pointer onto the freeDoneEvent vector and calls dmaDone() when triggered. While most of the time the burst length of a transaction will be the max burst length set by the driver, any trailing bytes must be handled with smaller lengths thus requiring the configurable burst length option.
