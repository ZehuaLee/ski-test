Detailed Description

EmulatedDriver is an abstract base class for fake SE-mode device drivers.

Specific drivers that allow applications to communicate with simulated hardware inside gem5 can be created by deriving from this class and overriding the abstract virtual methods.

Currently only open() and ioctl() calls are supported, but other calls (e.g., read(), write(), mmap()) could be added as needed.
