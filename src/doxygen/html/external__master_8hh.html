Detailed Description

ExternalMaster is a memory object representing a binding from a gem5 slave to a master port in a system external to gem5.

During initialisation, a `handler' for the port type specified in the port's port_type parameter is found from the registered port handlers provided with registerHandler. Once a handler is found, it is passed the port_data parameter of the port which can be used to identify the external port which is to be bound to. A port handler will usually construct a bridge object in the external system to accomodate the port-to-port mapping but this bridge is not exposed to gem5 other than be the presentation of the MasterPort which can be bound.

The external port must provide a gem5 MasterPort interface.
